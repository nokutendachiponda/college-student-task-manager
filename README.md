# Taskify | Academic Command Center 

Taskify is a sophisticated Academic Command Center designed to centralize student workflows into a high-performance, data-driven interface. By integrating real-time task management, course planning, and visual progress analytics, it solves the critical problem of academic fragmentation where students struggle to manage disparate deadlines across multiple platforms.

In an increasingly digital education industry, Taskify represents a shift toward personalized productivity ecosystems that empower learners with "Mission Control" visibility over their academic trajectory, ultimately improving time management and student outcomes through actionable insights.

![Taskify Banner](https://picsum.photos/seed/taskify/1200/400)

## Key Features

- **Cosmic Timeline:** A dynamic calendar view to track academic cycles and deadlines.
- **Mission Control Dashboard:** Visual analytics and progress tracking using Recharts.
- **Stellar Backlog:** Manage unscheduled tasks and quick-entry missions.
- **Academic Map:** Comprehensive course planner to organize your semester.
- **Real-time Sync:** Powered by Firebase for instant updates across devices.
- **Admin Panel:** System-wide broadcast capabilities and personnel management.
- **Modern UI/UX:** Built with Tailwind CSS and Framer Motion for a fluid, "space-age" experience.

## Tech Stack

- **Frontend:** React 19, TypeScript, Vite
- **Styling:** Tailwind CSS 4.0
- **Database & Auth:** Firebase (Firestore & Google Authentication)
- **Animations:** Framer Motion
- **Charts:** Recharts
- **Icons:** Material Symbols & Lucide React
